# Jolly Cabs | Premium Cab Service in Hyderabad

![Jolly Cabs](https://images.unsplash.com/photo-1494976388531-d1058494cdd8?auto=format&fit=crop&q=80&w=800)

Welcome to the official repository for **Jolly Cabs**, a modern, premium, and fully responsive landing page designed for a high-end cab service based in Hyderabad, India.

## 🚀 Features

- **Premium UI/UX**: Dark mode aesthetic with gold/yellow accents (`#facc15`) and smooth glassmorphism effects.
- **Fully Responsive**: Pixel-perfect design that scales flawlessly from ultra-wide desktop monitors down to the smallest mobile devices (320px).
- **Interactive Elements**: Scroll-reveal animations (powered by AOS), live statistics counters, and premium glowing hover effects on cards.
- **WhatsApp Integration**: Direct "Book Now" buttons that dynamically link to a WhatsApp lead generation URL.
- **Modern Development**: Built entirely using pure HTML5, CSS3, and Vanilla JavaScript. No heavy frameworks!

## 📁 Project Structure

- `index.html`: The main structural layout including navigation, hero, fleet grid, about us, services, and footer.
- `style.css`: Comprehensive design system with CSS variables, fluid typography, and extensive media queries for responsive behavior.
- `script.js`: Handles mobile menu toggling, smooth scrolling, and the animated statistics counter.
- `assets/images/`: Local repository for high-quality vehicle and background placeholder images.

## 🛠️ Setup & Usage

To view this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/bharathkpd/jollycabs.git
   ```
2. Navigate to the project directory:
   ```bash
   cd jollycabs
   ```
3. Open `index.html` in any modern web browser or serve it using a local live server (e.g., VS Code Live Server or Python HTTP Server):
   ```bash
   python -m http.server 8000
   ```

## 🌐 Publishing

This website is designed as a static HTML/CSS/JS package and is 100% ready to be deployed on **GitHub Pages**, **Netlify**, or **Vercel** with zero configuration required.
